# ChargerMap by FLUI ⚡

O ChargerMap by FLUI é um aplicativo mobile desenvolvido para auxiliar motoristas de veículos elétricos na busca por estações de carregamento de forma prática, intuitiva e moderna.

O projeto foi desenvolvido como parte do Charge Platform Challenge, proposto pela FIAP, com foco em mobilidade elétrica, experiência do usuário e sustentabilidade.

---

# 📱 Funcionalidades

- Mapa interativo com pontos de recarga;
- Visualização detalhada dos postos;
- Filtros de busca;
- Interface moderna e responsiva;
- Navegação entre telas;
- Identidade visual inspirada em mobilidade elétrica.

---

# 🎨 Identidade Visual

## Paleta de cores
- `#B947FA`
- `#555799`

A proposta visual busca modernizar o conceito tradicional de mobilidade elétrica, normalmente associado ao verde e azul escuro, trazendo tons de roxo sem perder a conexão com tecnologia, sustentabilidade e inovação.

## Tipografia
- **Hanken Grotesk** → títulos e destaques
- **Poppins** → textos e interface

# ▶️ Como executar o projeto

## 1. Clone o repositório

```bash
git clone LINK_DO_REPOSITORIO
```

---

## 2. Acesse a pasta do projeto

```bash
cd chargermap-app
```

---

## 3. Instale as dependências

```bash
npm install
```

---

## 4. Execute o projeto

```bash
npx expo start --tunnel
```

---

## 5. Abrindo no celular

- Instale o aplicativo Expo Go;
- Escaneie o QR Code exibido no terminal;
- Caso necessário, utilize o link `exp://`.
